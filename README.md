# 🎮 Pulse Jump – AI Generated Game (CrewAI)

## 🧠 Overview

Pulse Jump is a one-tap minimal game where jump mechanics dynamically change during gameplay.

This project uses a **multi-agent system (CrewAI)**:

* 🎨 Game Designer Agent → creates mechanics
* 💻 Senior Engineer Agent → builds the game
* 🧪 QA Engineer Agent → tests and improves gameplay

## ⚙️ Tech Stack

* Python
* CrewAI
* Google Colab

## 🎮 Features

* Dynamic jump behavior
* Procedural difficulty
* AI-driven game iteration

## 🤖 Multi-Agent Workflow

Game Design → Code Generation → Testing → Improvement Loop
